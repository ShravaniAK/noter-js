About

The Note Maker

Noter Js is a web app for taking notes. It is made using HTML, CSS, and JS . It is convenient and easy to use, and its search feature helps the user quickly look for a particular note. Use Noter Js to take notes on anything intriguing, from a reminder to a to-do list.

Features of this application :

    Detele your notes, Save your notes ,Search them & Edit your notes.
